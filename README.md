# life-manager
This app is a business or daily task management application. It includes useful features such as a search function to determine if something is beneficial or harmful. This is the first version, and the app is still under development.
هذا التطبيق مخصص لإدارة الأعمال أو المهام اليومية. يتضمن ميزات مفيدة مثل خاصية البحث لتحديد ما إذا كان شيء ما مفيدًا أم ضارًا. هذه هي النسخة الأولى، والتطبيق لا يزال قيد التطوير.
